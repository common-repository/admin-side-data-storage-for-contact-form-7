This plugin will helps you to store and manage data of Submitted Contact Form 7 entries.

== Description ==

This plugin will store data which is submitted by Contact form 7 and it will provide various features to manage it with different ways.

== Features ==

- To work with this initially user needs active version of Contact form 7.

- All submitted entries of Contact form 7 will be display.

- User can perform various actions like bookmark, read/unread, export data into PDF, CSV and Excel format.

- User can use datewise filter entries of Contact form 7.

- User can display submitted entries on Frontend by using Shortcode.

== Installation ==

1. Upload the `asds-cf7` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Sidebar of Admin Panel and Plugin name will show, by clicking on that user can configure the plugin base on their need.
4. Once all configuration completes plugin will start its work.
